# 用户确认模板

## 洗护方案确认

```
推荐方案
- 门店：{门店名}
- 服务：{服务名}
- 时间：{日期} {时间}
- 接宠：{接宠时间} 上门
- 送回：约 {送回时间}
- 预计费用：{服务价格} 元{活动折扣说明} + 宠物专车费

是否确认？
```

### 示例

```
推荐方案
- 门店：泡泡宠物生活馆 南山店
- 服务：狗狗洗澡 + 刷牙
- 时间：今天 16:00
- 接宠：15:30 上门
- 送回：约 17:30
- 预计费用：139 元（工作日9折后 125.1 元） + 宠物专车费

是否确认？
```

## 商品订单确认

```
推荐订单
- 商品：{商品名}
- 数量：{数量} {单位}
- 门店：{门店名}
- 预计送达：{送达时间}
- 预计费用：{商品价格} 元{活动折扣说明} + 同城配送费

是否确认下单？
```

### 示例

```
推荐订单
- 商品：成猫粮 5kg
- 数量：1 袋
- 门店：泡泡宠物生活馆 南山店
- 预计送达：2 小时内
- 预计费用：199 元 + 同城配送费

是否确认下单？
```

## 改约确认

```
预约变更
- 原方案：{原服务} @ {原门店} {原时间}
- 新方案：{新服务} @ {新门店} {新时间}
- 费用变化：{差价说明}
- 宠物专车：{新接送安排}

是否确认变更？
```

## 追加服务确认

```
追加服务
- 原服务：{原服务名}
- 追加：{追加服务名}
- 额外耗时：{分钟} 分钟
- 差价：+{金额} 元{活动折扣说明}
- 新总时长：{分钟} 分钟
- 新总费用：{金额} 元

是否确认追加？
```

## 取消确认

```
确认取消
- 预约：{服务名} @ {门店名} {时间}
- 宠物专车：{接送安排}
- 取消后宠物专车将同步取消

确认取消预约？
```

## 日程冲突提示

```
日程冲突
- {时间} 已有日程：{冲突事件名}
- 推荐替代时间：
  1. {替代时间1}
  2. {替代时间2}

是否选择替代时间？
```

## 库存不足提示

```
库存不足
- {商品名} 在 {门店名} 暂无库存
- 替代方案：
  1. {其他门店名} 有库存 {数量} {单位}，需同城配送
  2. 替代商品：{替代商品名}，{价格} 元

请选择替代方案？
```
