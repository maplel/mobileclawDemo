# AI Log 模板

## 洗护预约场景

### 查询阶段

```
{时间} 检查到{宠物名}距离上次洗澡已 {天数} 天。
{时间} 已查询附近 {数量} 家宠物店。
{时间} {门店名}今天 {时间} 可预约"{服务名}"。
{时间} 检查用户日程：{时间} 无冲突。
{时间} 等待用户确认服务方案。
```

### 预约确认阶段

```
{时间} 用户已确认服务方案。
{时间} 已创建预约：{服务名} @ {门店名} {时间}，预约编号 {booking_id}。
{时间} 已安排宠物专车：{接宠时间} 上门接，{送回时间} 送回。
{时间} 预约完成。
```

### 改约阶段

```
{时间} 用户要求改约。
{时间} 已查询新档期：{新时间} 可用。
{时间} 预约已变更：{变更项}，新时间 {新时间}。
{时间} 已更新宠物专车时间。
```

### 取消阶段

```
{时间} 用户确认取消预约。
{时间} 已取消预约 {booking_id}。
{时间} 已取消宠物专车。
{时间} 已删除日历事件。
```

## 商品购买场景

### 查询阶段

```
{时间} 收到用户需求：购买{商品类别}。
{时间} 已查询常买品牌：{商品名}。
{时间} {门店名}库存 {数量} {单位}，可 {时间} 内同城配送。
{时间} 等待用户确认商品和送达地址。
```

### 下单确认阶段

```
{时间} 用户已确认订单。
{时间} 已创建订单：{商品名} x{数量}，订单编号 {order_id}。
{时间} 已安排同城配送，预计 {送达时间} 送达。
{时间} 订单完成。
```

### 送达后

```
{时间} 订单 {order_id} 已送达。
{时间} 已设置 {间隔}天后补货提醒。
```

## 异常场景

```
{时间} {门店名} {日期} 闭店，已推荐替代方案。
{时间} {门店名} {日期} 已约满，已推荐替代时间。
{时间} {商品名} 在 {门店名} 暂无库存，已推荐替代方案。
{时间} 暂无宠物专车可用，建议自行前往。
{时间} 同城配送暂不可用，建议到店自取。
{时间} {时间} 存在日程冲突：{冲突事件}，已推荐替代时间。
{时间} 操作失败：{步骤}，原因：{错误信息}。
```
